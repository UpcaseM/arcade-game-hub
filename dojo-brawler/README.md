# Dojo Brawler

A lightweight static arcade fighting game built with HTML5 canvas.

## Controls

- Desktop movement: `WASD` or arrow keys
- Desktop attack: `Space`, `J`, or `K`
- Mobile movement: left virtual joystick
- Mobile attack: right Attack button
- Restart after round end: `Restart` button (`R` key also works)
- Back to hub: `../` via in-game buttons

## Features

- Player movement + melee attack loop
- Enemy AI (approach, strafe, retreat, attack)
- Hurtbox/hitbox collision checks with knockback and invulnerability window
- HP bars and round end state (`You Win` / `You Lose`)
- Restart and Back to Hub actions
- Relative paths for GitHub Pages subpath compatibility
